# What you will learn in this course?

1. You will write kubernetes manifests with confidence after going through live template writing sections

2. You will learn 30+ kubernetes concepts and use several AWS Services in combination with EKS

3. You will learn Kubernetes Fundamentals in both imperative and declarative approaches

4. You will learn writing & deploying k8s manifests for storage concepts like storage class, persistent volume claim pvc, mysql and EBS CSI Driver.

5. You will learn to switch from native EBS Storage to RDS Database using k8s external name service

6. You will learn writing and deploying load balancer k8s manifests for Application and Network load balancers

7. You will learn writing ingress k8s manifests by enabling features like context path based routing, SSL, SSL Redirect and External DNS.

8. You will learn writing k8s manifests for advanced fargate profiles and do mixed mode workload deployments in both EC2 and Fargate Serverless

9. You will learn using ECR - Elastic Container Registry in combination with EKS.

10. You will implement DevOps concepts with AWS Code Services like CodeCommit, CodeBuild and CodePipeline

11. You will implement microservices core cocepts like Service Discovery, Distributed Tracing using X-Ray and Canary Deployments

12. You will learn to enable Autoscaling features like HPA,VPA and Cluster Autoscaler

13. You will learn to enable monitoring and logging for EKS cluster and workloads in cluster using CloudWatch Container Insights

14. You will learn Docker fundamentals by implementing usecases like download image from Docker Hub and run on local desktop and  build an image locally, test and push to Docker Hub.

15. You will learn helm & helm chart.

16. You will master many kubectl commands over the process
